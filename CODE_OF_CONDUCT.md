# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## TL;DR

- Be respectful and constructive in discussions.
- Welcome newcomers and help them learn.
- Focus on the idea, not the person.
- Assume good faith. Ask for clarification before escalating.
- No personal attacks, no demeaning language, no off-topic politics in issues/PRs.

## Scope

This applies to all project spaces — issues, pull requests, commit messages, review comments, and any direct communication about the project.

## Reporting

If you see behavior that violates this, please email the maintainer at `terziogluyusuff1@gmail.com` with subject `[react-native-advanced-date-picker CoC]`. Reports are confidential.

## Enforcement

The maintainer may remove comments, close issues, or block users that violate this policy. For repeated or severe violations, access may be revoked permanently.

For the full text and examples, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
